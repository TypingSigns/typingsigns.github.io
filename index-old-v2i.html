Typing Signs

A collection of personal ASL projects

ASL Text (iOS)

This app is intended for quickly jotting down notes in ASL.

It uses a colored version of the SignWriting font.

Updated: 2018-08

ASL Keyboard (Experimental)

This keyboard allows one to quickly compose ASL sentences by typing english glosses.

These can consist of static images (SignWriting), or animated images (from SpreadTheSign or SignWithRobert). Sentences can then be copied to Microsoft Word etc.

Created to work on desktop and mobile (iPhone).
Updated: 2018-01.
